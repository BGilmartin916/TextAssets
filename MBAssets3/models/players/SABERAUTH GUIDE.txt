// SABERAUTH.CFG SHOULD NEVER EXCEED 2048 CHARACTERS

// This configuration file is used to set the default lightsaber hilts and colors for a character with the authentic sabers button in the hilt select menu.

// Authentic sabers may be defined in the format:
	// hilt hue (single)
	// hilt hue (special)
	// hilt hue (special2)
// Or alternatively:
	// hilt (single)
	// hue (single)
	// hilt (special)
	// hue (special)
	// hilt (special2)
	// hue (special2)
	
// To configure a staff hilt, set the second special model to none.

// To define default sabers for any of a model's skins:
	// all_sabers
	// single_1 0.5555 
	// dual_2 0.5555 
	// none 0.5555
// To define sabers for a specific skin (overrides default if set):
	// sabers_skinname
	// single_1 0.5555 
	// single_1 0.5555 
	// single_2 0.5555
