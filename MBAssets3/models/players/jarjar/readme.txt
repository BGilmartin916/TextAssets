LODS                    : NOT YET


* Construction *
Poly Count              : 3082 Polys
Vert Count              : 2021 Verts
Skin Count              : 6 Skins  (Default and CTF for Jar Jar, Default and CTF for Gungan) 
Base info               : Mesh and skin created from scratch.  Ravens animations were used.
                          Sounds taken from from TPM, or the TPM game.
Software used           : 3D Studio Max 4.2, Paintshop Pro 7, Assimilate, Modview, Cool Edit
Known Bugs              : Some minor clipping of the ears and his clothes, but nothing that 			  I feel detracts from the model in a huge way.
Build/Animation time    : Created over a period of about 3 weeks


* How to use this model *

Extract the mc_jarjar.pk3 file into the jk2/gamedata/base/ directory.
